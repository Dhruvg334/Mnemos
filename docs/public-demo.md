# Public demonstration

The public dashboard is a read-only product tour backed by synthetic industrial records. It is intentionally available without authentication so evaluators can inspect the complete navigation and evidence model.

## Guest access

Guests may inspect assets, documents, graph relationships, investigation results, compliance states, expert knowledge, and agent traces. They may use scripted demonstration interactions that do not mutate backend state.

## Protected actions

Authentication remains required for uploads, deletions, password changes, membership management, organisation changes, query cancellation/retry, approval decisions, and governed domain actions. The backend enforces these rules independently of frontend modals.

## Data policy

Names, organisations, documents, findings, and measurements shown publicly are synthetic. Real workspace records must never be copied into the demonstration adapter or repository screenshots.
